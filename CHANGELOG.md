# Change Log

All notable changes to the Soma extension will be documented in this file.

## [0.0.1] - 2026-01-05

### Added
- Initial release of Soma language support
- Integration with SouLS (Soma Language Server)
- Syntax highlighting with TextMate grammar
- Semantic token highlighting from LSP
- IntelliSense features:
  - Code completion with type signatures
  - Hover information for symbols
  - Go to definition across files
  - Find all references
  - Document symbols and outline view
- Real-time diagnostics (errors and warnings)
- Language configuration:
  - Auto-closing brackets and quotes
  - Comment toggling support
  - Smart indentation rules
- Commands:
  - "Soma: Restart Language Server" - Restart the language server
  - "Soma: Show Output" - Show language server output channel
- Automatic project discovery for haoma projects
- Cross-module type checking and navigation
- Support for .soma file extension
